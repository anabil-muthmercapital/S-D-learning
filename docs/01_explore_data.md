# 01 — استكشاف البيانات (Explore Data)

## الهدف
قبل أي تحليل، لازم تشوف البيانات بعينك. الـ notebook ده مالوش أي علاقة بكشف المناطق — هو بس عشان تتعرّف على شكل البيانات اللي كل الكود هيشتغل عليها.

## ليه أول واحد؟
لو رحت تقرأ `detect_zones()` قبل ما تشوف البيانات، هتبقى بتقرأ كود بيتعامل مع حاجة مش شايفها. أول خطوة دايماً: افهم المُدخَل.

## هتعمل إيه
1. تفتح `fixtures.csv` بـ pandas.
2. تبص على الأعمدة: `open, high, low, close, volume`.
3. تشوف كام صف (98 شمعة) وإزاي الـ index (تواريخ أسبوعية).
4. تفتح `fixtures_labeled.csv` كمان وتقارن — ده نفس البيانات بس فيه عمود `scenario` و `note` بيقولك كل شمعة هي إيه.

## الكود اللي تكتبه بإيدك
```python
import pandas as pd

# البيانات النضيفة اللي الكود بياكل منها
df = pd.read_csv("../fixtures.csv", index_col=0, parse_dates=True)
print(df.shape)          # (98, 5)
df.head(20)

# ورقة الإجابات — نفس البيانات + شرح كل شمعة
labeled = pd.read_csv("../fixtures_labeled.csv", index_col=0, parse_dates=True)
labeled[["scenario", "note"]].head(30)
```

## السؤال اللي بيقفله
- إيه الفرق بين `fixtures.csv` و `fixtures_labeled.csv`؟
- كام سيناريو موجود؟ (جرّب `labeled["scenario"].unique()`)
- ليه فيه شموع اسمها "warmup" في الأول؟
